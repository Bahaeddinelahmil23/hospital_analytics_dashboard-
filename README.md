# 🏥 Hospital Analytics Dashboard (Healthcare Data)

## 📌 Overview
This project presents an interactive healthcare analytics dashboard designed to monitor hospital performance, patient behavior, and financial operations.

The dashboard transforms raw healthcare data into meaningful insights, helping improve decision-making in hospital management, patient care, and financial planning.

---

## 🎯 Objectives
- Analyze patient flow and hospital service usage  
- Evaluate patient satisfaction and quality of care  
- Monitor hospital revenue and cost structure  
- Understand patient demographics and distribution  
- Identify trends in hospital visits over time  

---

## 📊 Dashboard Features

- **Patient Metrics**
  - Average length of stay  
  - Patient satisfaction score  
  - Total number of visits  

- **Financial Analysis**
  - Billing amount  
  - Treatment cost  
  - Medication cost  
  - Room charges  

- **Patient Insights**
  - Distribution by age groups  
  - Number of patients by department  
  - Service type analysis (Emergency, Inpatient, Outpatient)  

- **Geographical Analysis**
  - Billing amount by city and region  

- **Operational Insights**
  - Payment status (Paid vs Pending)  
  - Monthly trends in hospital visits  

---

## 🛠 Tools & Technologies
- **Power BI** → Dashboard design & visualization  
- **Excel** → Data preprocessing  
- **DAX** → Custom measures and KPIs  

---

## 📈 Key Insights
- Emergency services have the highest average length of stay  
- Older patients (65–79) represent the largest group  
- Cardiology and General Surgery handle the highest patient volume  
- A significant portion of payments remains pending  
- Revenue varies across different cities and regions  

---

## 🚀 Impact
This dashboard helps healthcare organizations:
- Improve resource allocation  
- Enhance patient care and satisfaction  
- Monitor financial performance  
- Identify operational inefficiencies
